# Test Notification Sound

Play the notification sounds to verify they're working correctly.

Run these commands to test:

1. **Task completion sound (Glass):**
```bash
afplay /System/Library/Sounds/Glass.aiff
```

2. **Attention needed sound (Funk):**
```bash
afplay /System/Library/Sounds/Funk.aiff
```

Report to the user whether the sounds played successfully.
